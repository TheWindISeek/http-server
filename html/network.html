Welcome to Computer Networking

This page provides an overview of computer networking concepts.

Introduction

Computer networking is the practice of linking computers together to exchange information and to share resources. Each computer in a network is called a node.

Types of Networks

  • Local Area Network (LAN)
  • Wide Area Network (WAN)
  • Metropolitan Area Network (MAN)

Local Area Network (LAN)

A LAN is a network that connects computers within a relatively small area such as a single building or a group of buildings.

Wide Area Network (WAN)

A WAN is a network that covers a broad area, such as connecting cities or countries.

Metropolitan Area Network (MAN)

A MAN is a network designed to connect multiple locations within a metropolitan area, such as a city.

Networking Protocols

Networking protocols define the rules for communication between devices on a network. Some common protocols include:

  • TCP/IP (Transmission Control Protocol/Internet Protocol)
  • HTTP (HyperText Transfer Protocol)
  • FTP (File Transfer Protocol)

Network Hardware

Hardware used in networking includes:

  • Routers
  • Switches
  • Hubs

Conclusion

Computer networking is a fundamental aspect of modern computing, enabling communication and resource sharing across various types of networks.
